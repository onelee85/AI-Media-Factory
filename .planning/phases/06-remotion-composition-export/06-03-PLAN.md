  <action>
    Create a lightweight React app in `remotion/player/` that embeds the @remotion/player.

    **remotion/player/index.html:**
    Standard HTML5 boilerplate with `<div id="root">` and `<script type="module" src="./index.tsx">`.

    **remotion/player/index.tsx:**
    ```tsx
    import { createRoot } from 'react-dom/client';
    import { App } from './App';
    createRoot(document.getElementById('root')!).render(<App />);
    ```

    **remotion/player/App.tsx:**
    ```tsx
    import { Player } from '@remotion/player';
    import { VideoComposition } from '../src/VideoComposition';
    ```

    The App component should:
    1. Read props from URL query parameters:
       - `?audio=...` — audio file URL
       - `?subtitles=...` — SRT text (URL-encoded or base64)
       - `?images=...` — comma-separated image URLs
       - `&title=...` — optional title text
    2. Default to placeholder props if no query params provided
    3. Embed the Player with:
       - `component={VideoComposition}`
       - `compositionWidth={1920}`, `compositionHeight={1080}`
       - `fps={30}`
       - `durationInFrames={calculated from subtitle data + buffer}`
       - `inputProps={parsedProps}`
       - `controls={true}`
       - `style={{ width: '100%', maxWidth: '960px', aspectRatio: '16/9' }}`
    4. Simple dark background (#1a1a2e) with centered player
    5. Show a header "Video Preview" above the player

    This will be a standalone HTML app. For development it can be served directly. For production it can be bundled with esbuild or served as-is.
